# AutoMakeBlank
这个小玩意可以在英语文章中随机挖空。
你可以用它来与你已有朗读音频的英语文章帮助你练习听力
或者搞点其他的qwq

# 注意事项
❗使用时需将文章本体复制到article.txt中
❗wordBlkList.txt是挖空**黑名单**，你可以将不需要的单词放到里面来避免它被挖空
    wordBlkList.txt使用方式：以英文逗号间隔单词，或者以行间隔单词，二者皆可。
                             注意不要输入空格，单词之间仅需要逗号/空行即可
